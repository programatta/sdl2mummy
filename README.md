# SDL2MUMMY
Remake del juego *Oh! Mummy* desarrollado en C++ con la librería SDL2 y para las siguientes plataformas:
* Desktop  
    * OSX
    * Linux
* Android
* iOS (wip)

## Notas.
Basado en la estructura de proyecto [**sdl2anddesk**](https://github.com/programatta/sdl2anddesk.git).
